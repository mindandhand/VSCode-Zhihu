# 支持说明

## 获取帮助

请通过 [GitHub Issues](https://github.com/mindandhand/VSCode-Zhihu/issues) 报告问题。提交前请先搜索是否已有相同问题，并尽量使用最新版本复现。

## Issue 应包含

- VS Code 版本、扩展版本和操作系统。
- 可以稳定复现问题的最短步骤。
- 预期行为与实际行为。
- 相关错误信息或已脱敏日志。
- 问题是否与登录、阅读、搜索、图片上传或发布有关。

如果问题涉及某个知乎页面，可以提供公开页面链接；不要提供私人草稿、未公开内容或账号数据。

## 隐私与安全

请勿在 Issue、Pull Request、截图或日志中提交以下信息：

- 完整 Cookie，尤其是 `z_c0`、`_xsrf`、`d_c0` 等字段。
- 密码、短信验证码、二维码登录信息或浏览器配置目录。
- 访问令牌、私有文章或草稿内容。

提交日志前，请将 Cookie、Authorization 请求头、账号 ID 和其他可识别信息替换为 `[REDACTED]`。如果已经公开泄露登录凭据，请立即删除内容并退出相关登录会话。

## 支持范围

本项目依赖知乎的非公开接口和页面行为，因此只能提供尽力而为的兼容维护。知乎接口、登录流程或风控策略变化可能导致部分功能暂时不可用。

项目不提供账号风控解除、验证码绕过、Cookie 获取之外的账号代操作，也不保证第三方接口长期稳定。

安全漏洞请不要公开披露可直接利用的细节；先创建不含敏感信息的 Issue，请求维护者提供后续沟通方式。
